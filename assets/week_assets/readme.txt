This is where you should put your week directories for images, sounds and music (not to confuse with "songs").

Examples:
assets/week_assets/week1/
assets/week_assets/weekCosmo/
assets/week_assets/<customWeekName>/

Once compiled, the folders will be moved to assets/ and will look like this:
assets/week1/
assets/weekCosmo/
assets/<customWeekName>/

Check assets/base_funkin/week7/ for a good example of the file structure
